# Info Panel - Getting Started

## Overview

Simple onboarding: 
1. User uploads their CV
2. Chat asks what kind of jobs they're looking for
3. Preferences are saved

That's it.

## What Gets Saved

```typescript
interface UserData {
  cv: {
    fileName: string;
    blob: Blob;
    textContent: string;  // Parsed for AI
    uploadedAt: Date;
  };
  
  preferences: {
    rawChat: string;      // The conversation
    parsed?: {            // AI-extracted preferences (optional)
      jobTypes?: string[];
      locations?: string[];
      salary?: string;
      mustHave?: string[];
      dealbreakers?: string[];
    };
  };
}
```
